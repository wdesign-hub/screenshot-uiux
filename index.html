1 Install Requirements

Before running the crawler, ensure you have Python installed on your system. Then, open your terminal and install the required libraries:

pip install selenium tqdm

Note for Mac users: If the command above fails, try using pip3 install selenium tqdm instead.

2 Run the Crawler

You can run the script from anywhere on your computer using the drag-and-drop feature in your terminal:

  1. Open your Terminal (Mac) or Command Prompt (Windows).
  2. Type python (make sure to include the space at the end).
  3. Drag and drop the crawler.py file directly into the terminal window.
  4. Press Enter.

3 The Automated Workflow

Once the script starts, it will guide you through the capture process:

Enter the starting URL (e.g., https://example.com): https://webflow.com
Enter the maximum number of pages to capture: 15

Native Folder Picker: After entering the limit, a native OS window will pop up automatically. Use it to select your destination folder or create a new one to keep your screenshots organized.

Under the hood, the crawler will:

🕸️ Extract internal links.
🎞️ Scroll to trigger animations.
🧹 Delete sticky popups.
📸 Capture CDP screenshots.
