1 Environment Setup

Ensure Python is installed, then pull the required dependencies via your terminal:

pip install selenium tqdm

Note for macOS: If the environment fails, alias to Python 3 using pip3 install selenium tqdm.

2 Initialize Crawler

Launch the crawler globally using your terminal's drag-and-drop pathing.

  1. Launch Terminal (macOS) or Command Prompt (Windows).
  2. Type python (include the trailing space).
  3. Drag and drop the crawler.py file directly into the terminal window.
  4. Press Enter to execute.

3 The Capture Pipeline

The script initiates an interactive session to define the scope of the audit:

Enter the starting URL (e.g., https://example.com): https://webflow.com
Enter the maximum number of pages to capture: 15

Native Folder Picker: A system OS window will trigger automatically to select or create your destination directory.

🕸️ Topology Mapping

Recursively extracts internal links up to your defined limit.

🎞️ Scroll & Settle

Forces lazy-loaded DOM assets and GSAP animations to render.

🧹 DOM Cleansing

Detects and destroys sticky cookie banners before snapping.

📸 CDP Captures

Executes native Chrome protocols for stitch-free images.
